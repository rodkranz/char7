Welcome to Char7.

This program has been made to help people to replace some ASCII code to HTML codes.

The Main Purpose

This program has already been made in PHP, but it was one of the first codes that I developed so, now I would like to create something better, creating to restore back, restore those backups and customize with things that I want to replace. I have developed the new Char7 (or c7) using GOlang.
